# 샘플 데이터셋 EDA 리포트

## 1. 개요

본 리포트는 대회 연습용으로 생성한 샘플 데이터셋에 대한 탐색적 데이터 분석(EDA) 결과를 정리한 문서입니다.

### 데이터셋 정보
- **샘플링 방법**: Stratified Sampling (카테고리 분포 유지)
- **Train 데이터**: 15,000개 대화
- **Validation 데이터**: 1,875개 대화
- **전체**: 16,875개 대화
- **Train/Val 비율**: 8.0:1
- **원본 대비 샘플링 비율**: 약 5.4% (279,992개 → 15,000개)

## 7. 모델링 시 고려사항

### 7.1 데이터 크기
- 샘플 데이터는 **15,000개**로 대회 데이터와 유사한 규모
- 빠른 실험과 반복 학습에 적합

### 7.2 불균형 문제
- **성별 불균형**: 여성 84.5% vs 남성 15.5%
  - 성별 기반 편향 가능성 주의
- **연령 불균형**: 20대 79%로 집중
  - 연령대별 성능 차이 발생 가능
- **카테고리 불균형**: 개인및관계(25.4%) vs 시사교육(4.7%)
  - Class imbalance 처리 필요 (가중치, oversampling 등)

### 7.3 발화 길이
- 평균 10자의 짧은 발화
- 짧은 문맥에서도 잘 작동하는 모델 선택 필요

### 7.4 추천 전략
1. **카테고리 균형**: Weighted loss 또는 focal loss 사용
2. **데이터 증강**: Back-translation, 패러프레이징
3. **Transfer Learning**: 한국어 사전학습 모델 활용 (KoBART, KoGPT 등)
4. **평가 지표**: 전체 accuracy뿐만 아니라 카테고리별 F1-score 확인
